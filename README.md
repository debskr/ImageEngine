# ImageEngine
A standalone minimal application that uses Pollinations AI API to generate images in different resolutions using Flux, Flux Turbo and ZImage models

# Instruction
1. Collect your API key from https://enter.pollinations.ai/
2. Open fluxgen.py file in notepad or any other code editor and paste your API key in API_KEY line and save
3. Run fluxgen-run.bat file
4. All the generated images will be saved in "Output" folder in the same application directory
